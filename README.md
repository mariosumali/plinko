# Plinko Game

A modernized Plinko game built with Python and Pygame. Features a dark, premium UI, real-time statistics, and physics-based gameplay.

## Created by Mario Sumali

## Features

- Physics-based ball movement
- Real-time Profit/Loss and Histogram charts
- Continuous ball streaming (Hold Spacebar)
- Custom synthesized sound effects
- Wide-screen layout (2400x1080)
- Interactive betting system

## Installation

1. Ensure you have Python installed.
2. Install the required dependencies:

```bash
pip install pygame numpy
```

## How to Run

Run the main script to start the game:

```bash
python plinko.py
```

## Controls

- Spacebar: Drop a single ball (Hold to stream balls)
- Mouse: Interact with UI elements (Bet Amount, Wallet)
- R: Reset the game board and statistics

## Sounds

The game includes a sound generator script (`generate_sounds.py`) that creates custom WAV files for clicks, scores, and pings. These are generated automatically if missing, or can be regenerated by running the script manually.
